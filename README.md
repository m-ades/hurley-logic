# Logic Proof Practice App

An interactive static web app (client-side only) for practicing formal logic proofs using Hurley's notation. Built with React and powered by the LogicPenguin derivation engine. All validation happens locally in the browser - no backend required.


## Tech Stack

- **React 18**
- **Material UI**
- **Vite**
- **LogicPenguin**
